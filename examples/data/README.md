Run ./download_data.sh and choose 2 for amazon_2023 dataset and 1 for Gift_Cards to start.

```bash
% ./download_data.sh
Choose dataset
1) amazon_2018
2) amazon_2023
3) quit
   #? 2
   Choose variant
1) Gift_Cards
2) Software
3) Video_Games
4) Electronics
   #? 1

Dataset: amazon_2023
Variant: Gift_Cards

Downloading 2023 metadata...
Downloading 2023 reviews...
✅ Done.
```

You can switch up the variant to Software, Electronics, or Video_Games later.  This is needed for most of the examples.
